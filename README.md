## Course Builder Ai
CourseBuilder.ai Lets you build a course, Which contains Set of Modules and a engaging Quiz with just a single prompt. Useful application for professors/teachers who don't want to create their courses from scratch in a manual way but with ease.
Live at : https://course-builder-ai.vercel.app/
